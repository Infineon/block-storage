Block Storage Library

General Description

This library provides a convenient way to abstract an underlying memory by abstracting it and definining APIs to interact with the device.

Getting Started

This section provides steps for getting started with this library using the two implemented abstraction supported already.

  1. Include the block-storage library header in the application
    #include "mtb_block_storage.h"
  2. Initialize the block storage device Example initialization using the pre implemented device built on top of HAL NVM
    mtb_block_storage_t my_nvm_bsd;
    mtb_block_storage_nvm_create(&my_nvm_bsd);
    mtb_block_storage_nvm_create

  3. The library should now be ready to perform operations.
    • Program operation.
      uint8_t data[MAX_PROG_SIZE];
      uint32_t programSize = my_nvm_bsd->get_program_size(my_nvm_bsd->context, address);
      result = my_nvm_bsd->program(my_nvm_bsd->context, address, programSize, data);
      CY_ASSERT(result == CY_RSLT_SUCCESS);
